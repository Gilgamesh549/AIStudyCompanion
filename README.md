# AI Study Companion App

## Overview

This is a simple Android application that simulates an AI-powered study assistant.
Users can enter study notes or questions, and the app will generate a simulated AI response.

## Features

* Input study notes or questions
* Click "Ask AI" to generate a response
* Display AI-generated summary

## Technologies Used

* Kotlin
* Android Studio
* XML Layout

## How to Run

1. Clone the repository
2. Open in Android Studio
3. Run on emulator or device

## Note

The AI response is simulated and does not connect to a real AI model.
